# ADCM Poá — Site da Igreja

Site institucional da Igreja ADCM Poá (Assembleia de Deus).

## Stack
- React 19 + Vite
- Tailwind CSS
- react-icons

## Seções
- Hero
- Sobre
- Cultos
- Ministérios
- Escola de Teologia
- Localização
- Footer

## Desenvolvimento
```bash
npm install
npm run dev
```

## Variáveis de ambiente
Nenhuma variável de ambiente necessária — o site é 100% estático.

## Deploy
Aponta para o domínio principal: `adcmpoa.com`

> O botão "Já tenho acesso" na seção Teologia aponta para a URL da plataforma.
> Ajuste a constante `PLATAFORMA_URL` em `src/components/Teologia/Teologia.jsx`.
